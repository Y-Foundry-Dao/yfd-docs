# 🔧 Maintenance
_(post deployment)_

MVP
{: .label .label-blue }
No vault maintenance functions exist in MVP release.  Vaults operate “as is.”

To upgrade or perform maintenance on a vault, a new vault proposal must be submitted and new vault contracts must be deployed.
